# Habitly Legal - GitHub Pages

Documentos legales para la aplicación Habitly.

## Configuración de GitHub Pages

### Paso 1: Crear repositorio

1. Ve a [github.com/new](https://github.com/new)
2. Nombre del repositorio: `habitly-legal` (o el que prefieras)
3. Hazlo **público**
4. Crea el repositorio

### Paso 2: Subir archivos

Sube estos archivos al repositorio:
- `index.md` - Página principal
- `privacy.md` - Política de privacidad
- `terms.md` - Términos y condiciones

### Paso 3: Activar GitHub Pages

1. Ve a **Settings** → **Pages**
2. En "Source" selecciona **Deploy from a branch**
3. En "Branch" selecciona **main** y carpeta **/ (root)**
4. Click en **Save**

### Paso 4: Obtener URLs

Después de unos minutos, tus URLs serán:

```
https://TU-USUARIO.github.io/habitly-legal/
https://TU-USUARIO.github.io/habitly-legal/privacy
https://TU-USUARIO.github.io/habitly-legal/terms
```

### Paso 5: Configurar en Google Play Console

1. Ve a Google Play Console
2. Tu app → **Política de privacidad**
3. Ingresa la URL: `https://TU-USUARIO.github.io/habitly-legal/privacy`

## Personalización

Recuerda reemplazar en los archivos:
- `[TU_EMAIL_DE_CONTACTO]` → Tu email real
- `[TU_NOMBRE_O_EMPRESA]` → Tu nombre o empresa
